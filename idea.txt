อยากสร้าง AI service ขึ้นมา 1 ตัว
ทำหน้าที่ในการวิเคราะห์ข้อมูลเกี่ยวกับ One Piece Card Game มีความเข้าใจในกติกาของ One Piece Card Game
มีข้อมูลของการ์ดทุกใบ มีข้อมูลสถิติการใช้งาน deck ของผู้เล่น 

สามารถตอบคำถามวิเคราะห์ข้อมูลเกี่ยวกับ One Piece Card Game
สามารถจำลองสถานะการเล่นของ deck ต่างๆ ในปริมาณมากๆ เพื่อวิเคราะห์โอกาสแพ้ชนะ
สามารถวิเคราะห์ว่าควรเสริมการ์ดใบไหนเข้าไปในเด็คที่สนใจเพื่อรับมือกลับกับ deck ของผู้เล่นที่นิยมเล่นตามสถิติช่วงเวลานั้นๆ
สามารถวิเคราะห์ว่าควรเลือกใช้ deck ไหนเพื่อใช้ในการเล่นเพื่อรับมือกลับกับ deck ของผู้เล่นที่นิยมเล่นตามสถิติช่วงเวลานั้นๆ

ฉันอยากโปรเจ็คนี้เพื่อฝึกฝนการพัฒนา ai agent ใช้ความสามารถของ ai ในรูปแบบต่างๆ

เทคโนโลยีที่ใช้
python
Fast API เพื่อทำ api service
่langgraph/langchain เพื่อทำ AI agent เนื่องจาก ai ต้องตอบคำถามหลายแบบ อาจจะต้องพัฒนา ai agent หลายตัวให้ทำหน้าที่แยกกันชัดเจน เพื่อให้จำนาญเฉพาะเรื่องต่างกันไป
LLM model เบื้องต้นใช้ gemini

อนาคตจะมี feature อื่นๆเพิ่มเติม หรือ เสริมความสามารถของ ai agent เพิ่มเติม เช่น การทำ evaluation , observability, logging, monitoring, tracing, guardrails

เขียนทุกอย่างแยกเป็น module เพื่อให้สามารถปรับแต่งเพิ่มเติมได้ในอนาคต เช่น api service, ai agent, LLM model , data , test

สำคัญมาก ไม่ต้องทำให้ซับซ้อนเกินไป พยายามให้เข้าใจง่ายมากที่สุด เพราะโปรเจ็คนี้มีไว้เพื่อศึกษาและทำความเข้าใจการพัฒนา AI service 

สามารถดึงข้อมูลการ์ด จาก https://tcgcsv.com/ จะโหดลด้วยมือในตอนแรก หรืออทำระบบยิง REST API / Downloadable JSON ก็ได้

แบ่งการพัฒนาออกเป็น phase ต่างๆ เพื่อให้ฉันสามารถมี process ที่ชัดเจน
